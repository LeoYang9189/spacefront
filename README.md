# ZSPACEPLUS 官网

基于 Next.js 开发的 ZSPACEPLUS 官方网站。

## 技术栈

- Next.js 14
- React 18
- Tailwind CSS
- Framer Motion
- TypeScript

## 功能特点

- 响应式设计
- 现代化UI/UX
- 动画效果
- 图片预览
- 品牌展示
- 产品展示

## 本地开发

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build

# 启动生产服务器
npm start
```

## 部署

项目使用 Vercel 进行部署，自动化部署流程。

## 目录结构

```
src/
  ├── app/                 # 应用页面
  ├── components/         # 共享组件
  ├── styles/            # 样式文件
  └── types/             # TypeScript 类型定义
public/                  # 静态资源
```
